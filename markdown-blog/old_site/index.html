About this site

Hello, and welcome to my website. It isn't completed yet, but I plan to have it done in June 2020. Currently I'm focusing on creating a functional and beautiful framework for future use from scratch rather than content.

After finishing my junior year at Portland State studying Computer Science, I realized that I still had no idea how to create or design a website. This site represents a personal project to get more familiar with basic web design and improve my understanding of how the internet functions from the ground up.

While I originally intended to host this site from home using a Raspberry Pi 3B, it became clear that this wouldn't be an option for me without dealing with Comcast's expensive proprietary hardware. That being said, because I still intend to do everything "the hard way," this website is currently being hosted on a VPS from UpCloud.com using a LEMP stack. While I could have easily used one of the countless javascript frameworks meant to get a very pretty website up and running in just minutes, I instead decided to accept the challenge of doing things myself. That being said, I don't plan on a career in front end web development, but that doesn't mean I can't learn the basics on my own time.

Thank you for stopping by.
